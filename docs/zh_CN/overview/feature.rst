基本特性
========

AWTK并无独创功能，而是博采众长，自成一家：

#. 具有Qt的多平台。
#. 具有EMWin的小巧高效。
#. 具有TouchGfx/Embedded Wizard的酷炫。
#. 完全开源，商用免费。
